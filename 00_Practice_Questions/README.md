# 練習問題

このセクションでは、OCE-H試験対策のための練習問題を提供します。

## 練習問題の目的

- 試験形式に慣れる
- 知識の定着を確認する
- 弱点を特定する
- 試験対策の準備を整える

## 問題形式

OCE-H試験では以下の問題形式が使用されます：

- **多肢選択問題（Multiple-choice）**: 1つの正解を選択
- **複数選択問題（Multiple-selection multiple-choice）**: 複数の正解を選択
- **リスト構築問題（Build-list）**: 項目を順序付きリストに配置
- **マッチング問題（Matching）**: 項目を対応する項目とマッチング
- **ドラッグ&ドロップ問題（Drag-and-drop）**: 項目を適切な場所にドラッグ&ドロップ
- **ポイント&クリック問題（Point-and-click）**: 図やダイアログで適切な場所をクリック
- **ホットエリア問題（Hot-area）**: 図の特定の領域を選択

## 練習問題の使い方

1. **順番に学習**: 各セクションの内容を学習してから、対応する練習問題に取り組む
2. **時間を計る**: 実際の試験時間（135分）を意識して時間を計る
3. **解答を確認**: 解答と解説を読んで理解を深める
4. **復習**: 間違えた問題を復習し、関連するセクションを再学習する
